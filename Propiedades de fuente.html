Propiedades de fuente

Las propiedades CSS de las fuentes son las que permiten controlar el tamaño, el tipo, el grosor o el estilo de las letras, entre otras cosas. En la siguiente imagen se muestran las propiedades de las fuentes más destacadas.

Font-family

Las comillas en la propiedad Font-family se utilizan para encerrar nombres de fuentes que contienen más de una palabra o caracteres especiales, como espacios, números o símbolos que podrían confundirse con otras partes de la sintaxis CSS.

See the Pen Font family by Mauricio Mendoza Cedeño (@Mauricio-Mendoza-Cede-o) on CodePen.

Font-style

La letra cursiva es una versión especial de la fuente, mientras que la letra oblicua es la versión normal de la fuente con una ligera inclinación.

See the Pen font style by Mauricio Mendoza Cedeño (@Mauricio-Mendoza-Cede-o) on CodePen.

Font-variant

La propiedad font-variant en CSS se usa para especificar si queremos que el texto se muestre en letra pequeña o no.

See the Pen Untitled by Mauricio Mendoza Cedeño (@Mauricio-Mendoza-Cede-o) on CodePen.

Font-weight

La propiedad font-weight de CSS es la encargada de definir el grosor o peso de una fuente. Esto significa que determina la claridad o negrita del texto.

See the Pen Untitled by Mauricio Mendoza Cedeño (@Mauricio-Mendoza-Cede-o) on CodePen.

Font-size

La propiedad font-size se utiliza para establecer el tamaño de la letra.

See the Pen Untitled by Mauricio Mendoza Cedeño (@Mauricio-Mendoza-Cede-o) on CodePen.
